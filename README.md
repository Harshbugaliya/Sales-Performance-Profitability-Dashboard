# Sales-Performance-Profitability-Dashboard
A dynamic, interactive data visualization tool built to explore sales performance and profitability for a suppermarket store.

The sales performance and profitability dashboard is a visually engaging and analytical power bi report to help user explore and compare sales and profit in a year in all over India. The dashboard focuses on highlighting the states and cities that have highest sales and profit.
